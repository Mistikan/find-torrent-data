# О программе
Эта программа для сопоставления файлов на носителе и файлов внутри торрент-файла.

Предположим, что у вас есть торрент-файл, который вы хотели бы поставить на раздачу, но вы не помните, где именно у вас лежат его файлы.
Программа поищет, проверит возможных кандидатов и сделает отчёт в виде json файла.
В дальнейшем отчёт можно применить в bash/python скриптах или посмотреть самому.

# Принцип работы программы
Программа работает в две стадии:
* составления базы данных файлов;
* сопоставление файлов в торрент-файле и базе данных.

## Составление базы данных файлов
Программа сканирует указанные директории и сохраняет следующую информацию о файлах:
* полный путь;
* размер;
* хеш-сумма crc32.

Хеш-сумма требуется, чтобы не ставить в комбинации одинаковые файлы (читай ниже).

## Сопоставление файлов в торрент-файле и базе данных
Программа работает по следующему алгоритму:
1. Читает торрент-файл.
1. Формирует gbmp(GroupBlockMultiPiece) и gbmf(GroupBlockMultiFile) блоки.
1. Ищет файлы gbmp блоков.
1. Ищет файлы gbmf блоков.
